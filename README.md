2. Run an application with `node` CLI

        $ node ./src/index.js

# Pros and Cons of Redux

### WHEN NOT TO USE REDUX

- Right budget
- Small to medium-size apps
- Simple UI/data flow
- Static data

### PROPS

 - Easier Debugging
 - Easier Testing
 - Undo / Redo
 - Preserve Page State
 - Presist State

### CONS

 - Complexity
 - Verbosity
